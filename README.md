# JANSETU (Team Setup Guide)

Follow these steps after cloning the project to run it on your system.

---

## 1. Clone the Repository

```bash
git clone https://github.com/Maa8nya/JANSETU.git
cd JANSETU
--

## Backend Setup (Flask)

Go to backend folder:
```bash
cd backend
Create virtual environment:

python -m venv venv

Activate it:

Windows:

venv\Scripts\activate
Install dependencies:

pip install flask

Run backend:

python app.py
--

## Frontend Setup (React Native - Expo)

Go to frontend folder:

cd ../frontend

Install dependencies:

npm install

Start app:

npm start

👉 Scan QR using Expo Go app
